# Modèle de forge — Mise à niveau Netherite

Dans le vanilla, le modèle se trouve dans les anciennes cités. La version custom permet de le **fabriquer** en remplaçant le modèle par un **lingot de Netherite** :

```
D N D
D R D
D D D
```

| Symbole | Matériau |
|---|---|
| D | Diamant |
| N | Lingot de Netherite |
| R | Netherrack |

→ Donne **1 Modèle de forge — Mise à niveau Netherite**

!!! warning "Coût"
    Coûteux volontairement (1 lingot de Netherite), mais évite d'attendre RNG d'une ancienne cité. Tu peux toujours en trouver « gratis » en explorant les anciennes cités si tu préfères.
