# Armures de cheval

Impossibles à trouver en skyblock puisque les donjons sont vides. **Motif en H avec 7 matériaux** :

```
X . X
X X X
X . X
```

## Recettes

| Matériau (X) | Résultat |
|---|---|
| Lingot de fer | 1 Armure de cheval en fer |
| Lingot d'or | 1 Armure de cheval en or |
| Diamant | 1 Armure de cheval en diamant |
