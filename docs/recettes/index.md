# Recettes custom

Le skyblock rend la plupart des ressources difficiles à obtenir. Ces recettes custom permettent de fabriquer ce qui est normalement trouvable dans une survie classique. **Toutes sont réalisables sur ton établi**.

## Liste des recettes

<div class="grid cards" markdown>

-   :material-pickaxe: **[Minerais (motif en croix)](minerais.md)**

    Fabrique tous les minerais (charbon, fer, or, diamant, émeraude, cuivre, redstone, lapis-lazuli) en versions Overworld, Abîmes et Nether.

-   :material-horse: **[Armures de cheval](armures-cheval.md)**

    Motif en H pour les armures de cheval fer / or / diamant.

-   :material-shell: **[Armures de nautile](armures-nautile.md)**

    Motif en coquille d'escargot pour les armures de nautile cuivre / fer / or / diamant (+ Netherite via forge).

-   :material-fruit-grapes: **[Baies lumineuses](baies-lumineuses.md)**

    Recette sans forme : baie sucrée + poche d'encre luminescente.

-   :material-fire: **[Haut fourneau — conversions de pierre](haut-fourneau.md)**

    Diorite → Calcite · Andésite → Tuf.

-   :material-anvil: **[Modèle de forge Netherite](modele-forge-netherite.md)**

    Recette custom pour fabriquer le modèle de forge de mise à niveau Netherite (sinon uniquement disponible dans les anciennes cités).

</div>
