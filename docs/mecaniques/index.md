# Mécaniques spéciales

Au-delà des recettes, certaines ressources s'obtiennent via des mécaniques custom héritées de **CarpetSkyAdditions** (mod jsorrell) que nous avons portées et adaptées pour MC 1.21.11 et le format multi-joueur.

## Catégories

<div class="grid cards" markdown>

-   :material-spider: **[Mobs et créatures](mobs.md)**

    Mécaniques liées aux mobs : Blaze → Breeze, conversion Vex → Allay, drops creeper chargé, marchand ambulant à chameau, drops Warden, etc.

-   :material-cube: **[Blocs et ressources](blocs.md)**

    Mécaniques liées aux blocs : sable via érosion corail, diamant renouvelable, druse d'améthyste, mycélium, buisson, chorus, etc.

-   :material-sword-cross: **[Chambres des épreuves](chambres-epreuves.md)**

    Système strict CSA-Reborn : activation par bâton de Breeze obligatoire, loot rebalancé, challenges Tier 3 pour les ornements d'armure.

-   :material-disc-player: **[Disques musicaux](disques.md)**

    Drops creeper-via-squelette par boîte englobante de structure : Pigstep, Otherside, 11, Creator, Precipice, Fragment 5.

-   :material-treasure-chest: **[Drops spéciaux](drops.md)**

    Tessons de poterie, motif de bannière piglin, fèves de cacao, hurleur sculk préservé, etc.

</div>

## Note sur les noms

Tous les termes Minecraft utilisés dans ce wiki suivent les **noms vanilla français Java officiels** ([fr.minecraft.wiki](https://fr.minecraft.wiki)). Quelques pièges fréquents :

- **Breeze** (et pas « Brise ») — kept en anglais en FR Java
- **Mooshroom** (Champimeuh est l'ancien nom obsolète)
- **Ender Dragon** (kept en anglais en FR Java)
- **Coffre-fort funeste** (et pas « sinistre »)
- **Hautes herbes** (pluriel) · **Herbe aquatique** = seagrass · **Algue** = kelp
- **Araignée chevauchée** (et pas « Cavalier d'araignée »)
- **Fiole** = glass bottle · **Fiole d'eau** = water bottle
